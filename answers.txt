
Q1: Why might we consider using Git?

Answer: Git makes collaboration for a team efficient and easy.
	Members can all write and test their code before pushing	
	to the rest of the team. Additionally, it is easy to revert
	back to old versions of the code contained within Git.

Q2: What field could data science be applied to? Give an Example.

Answer: Data Science can be applied to the field of Finance. For instance
        using data science one could take past information/trends of 
	stocks and the stock market in order to predict future outcomes.

Q3: What is the difference between Merging and Rebasing?

Answer: Merging keeps track of the history between two branches, while
	rebasing applies all the commits of a branch to another, leaving
	one commit history.
